# 產品設計規格書 - 卡在國際通

## 1. 專案概述

### 1.1 專案名稱
**卡在國際通** (Stuck in Kokusai Street)

### 1.2 專案簡介
一個以沖繩國際通為中心的四天兩夜旅遊規劃前端網站，提供完整的行程安排、打包清單、實用旅遊資訊，協助旅客輕鬆規劃沖繩之旅。

### 1.3 目標用戶
- **主要用戶群**：年輕背包客（20-35歲）、情侶/蜜月旅客
- **次要用戶群**：首次前往沖繩的自由行旅客
- **用戶特徵**：
  - 喜歡使用手機瀏覽
  - 重視預算規劃
  - 需要詳細的交通指引
  - 希望發現在地美食

### 1.4 專案目標
- 提供清晰、實用的四天兩夜沖繩旅遊規劃
- 降低旅遊規劃的時間成本
- 提供互動式打包清單功能
- 支援 RWD 響應式設計，優化手機使用體驗

---

## 2. 技術規格

### 2.1 前端技術棧
- **框架建議**：React / Next.js / Vue.js / Nuxt.js（待技術選型）
- **CSS 框架**：Tailwind CSS / Bootstrap（待技術選型）
- **地圖整合**：Google Maps API
- **響應式設計**：RWD (Mobile-first design)
- **語言支援**：繁體中文 (預設)、英文

### 2.2 資料管理
- **類型**：全靜態內容
- **內容儲存**：直接寫在程式碼中或使用 JSON/Markdown 檔案管理內容
- **狀態管理**：LocalStorage（用於打勾清單的狀態保存）

### 2.3 第三方 API 整合
- **Google Maps API**：景點位置顯示、路線規劃
- **天氣 API**：Open Weather Map / Weather API
- **匯率 API**：ExchangeRate-API / Fixer.io

### 2.4 瀏覽器支援
- Chrome (最新版本)
- Safari (最新版本)
- Firefox (最新版本)
- Edge (最新版本)
- Mobile Safari (iOS 13+)
- Chrome Mobile (Android 9+)

## 4. 四天兩夜行程內容架構

### 4.1 Day 1: 抵達沖繩 - 國際通巡禮

**主題**：抵達、安頓、初探國際通

**行程概要**：
- 09:00 - 12:00：抵達那霸機場、入境、領取 WiFi
- 12:00 - 13:30：前往飯店寄放行李、午餐
- 14:00 - 17:00：國際通散步、購物
- 17:30 - 19:00：晚餐（國際通美食）
- 19:30 - 21:00：逛國際通夜市、土產店
- 21:30 - ：回飯店休息

**主要景點**：
1. 那霸機場
2. 國際通商店街
3. 牧志公設市場
4. 壺屋陶器街（備選）

**推薦美食**：
- 沖繩料理：暖暮拉麵、通堂拉麵、海鮮食堂
- 點心：Blue Seal 冰淇淋、御菓子御殿

**預算範圍**：¥5,000 - ¥8,000

**主要景點**：
1. 美麗海水族館
2. 海洋博公園

**推薦美食**：

- 陰影：0 2px 8px rgba(0,0,0,0.1)
- 圓角：12px
- Hover 效果：陰影加深、輕微上移

**打勾清單 (Checkbox)**：
- 未勾選：空心圓框
- 已勾選：藍色圓底 + 白色勾勾 ✓
- 已勾選項目文字：灰色 + 刪除線
- 動畫：勾選時有彈跳效果

### 5.6 圖示系統
使用圖示庫：**Feather Icons** 或 **Material Icons**

- 🏠 Home

## 6. 響應式設計 (RWD) 規範

### 6.1 Mobile 優先策略
設計時以手機版為主要考量，再往上擴展到平板和桌面版。

### 6.2 各裝置版型規劃

#### Mobile (< 768px)
- 單欄式佈局
- 漢堡選單
- 全寬卡片
- 堆疊式內容
- 觸控優化（按鈕最小 44x44px）
- 打包清單：展開/收合式分類
- 地圖：可切換全螢幕模式

#### Tablet (768px - 1023px)
- 2 欄式佈局（部分區域）
- 卡片開始並排
- 側邊選單或展開式選單
- 行程卡片：2 欄並排

#### Desktop (1024px+)
- 3-4 欄式佈局
- 完整導覽列
- Sidebar 固定導覽
- 行程卡片：3 欄並排
- 地圖與內容並排顯示
- 更大的圖片和更多的留白

### 6.3 關鍵互動優化

**Mobile 特殊考量**：
- 下拉刷新（可選）
- 底部導覽列（固定）
- 手勢操作（滑動切換 Day 1-4）
- 圖片延遲載入 (Lazy Loading)
- 減少動畫以提升效能

**觸控體驗**：
- 所有可點擊元素至少 44x44px
- 適當的點擊回饋（ripple effect）
- 避免 hover-only 互動
